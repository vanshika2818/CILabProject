# CI Lab Project: Jenkins Pipeline Implementation

## Overview
This repository contains a complete CI/CD pipeline implementation using Jenkins. It demonstrates how to automate the Build, Test, and Deployment phases for a Java-based application.

## Repository Structure
- `src/`: Contains Java source code and unit tests.
- `scripts/`: Automation scripts (Python/Shell) for build and deployment.
- `docker/`: Dockerfile for containerizing the application.
- `Jenkinsfile`: Defines the automated pipeline stages.
- `pom.xml`: Maven configuration for dependency management.

## Tech Stack
- **CI/CD:** Jenkins
- **Language:** Java
- **Build Tool:** Maven
- **Scripting:** Python, Bash
- **Containerization:** Docker
